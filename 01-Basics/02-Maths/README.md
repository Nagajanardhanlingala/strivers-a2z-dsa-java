# ⭐ Basic Maths

This folder contains Java solutions for the **Basic Maths** section from **Striver's A2Z DSA Sheet**.

## 📚 Problems Covered

- Count Digits
- Reverse Number
- Palindrome Number
- GCD / HCF
- Armstrong Number
- Print All Divisors
- Check Prime Number

## 🎯 Concepts Learned

- Digit extraction using `%` and `/`
- Number reversal
- Palindrome checking
- Euclidean Algorithm for GCD
- Armstrong number logic using `Math.pow()`
- Divisor pair optimization (`O(√N)`)
- Prime number checking (`O(√N)`)

## ⏱️ Time Complexities

| Problem | Time Complexity |
|---------|-----------------|
| Count Digits | O(log N) |
| Reverse Number | O(log N) |
| Palindrome Number | O(log N) |
| GCD / HCF | O(log(min(a, b))) |
| Armstrong Number | O(log N) |
| Print All Divisors | O(√N) |
| Check Prime Number | O(√N) |
